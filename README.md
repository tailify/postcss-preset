> Tailify's very own opinionated PostCSS preset.

## Installation

```
$ npm install --save-dev @tailify/postcss-preset
```

## Usage

To get started, add this to your `postcss.config.js` file:

```js
module.exports = require('@tailify/postcss-preset');
```
